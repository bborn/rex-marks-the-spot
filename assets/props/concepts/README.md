# Prop Concept Art

Pixar-style concept art for key props in "Fairy Dinosaur Date Night".

## Props

### Magic Minivan
The Bornsztein family's minivan that travels through time-warps.
- **v1**: Normal family minivan with messy interior
- **v2**: Magical version with blue time-warp energy
- **v3**: Damaged/crashed in jurassic swamp
- **v4**: Interior view showing family chaos

### Mop-Wand
Ruben the fairy godfather's magic wand (actually a janitor's mop).
- **v1**: Dormant, worn and shabby
- **v2**: Casting spell with glowing mop head
- **v3**: Detail view of magical runes and tape repairs
- **v4**: Floating with magical aura

### Stuffed Dino
Leo's beloved stuffed dinosaur toy (brontosaurus).
- **v1**: Full view, well-loved and worn
- **v2**: Detail showing loose button eye and patches
- **v3**: Sleeping pose
- **v4**: Action/play pose

### Family Photo
Framed portrait of the Bornsztein family.
- **v1**: Traditional studio portrait
- **v2**: Candid outdoor style
- **v3**: On fireplace mantle
- **v4**: Damaged/cracked from adventure

### Fairy Dust Jars
Magical containers for fairy dust.
- **v1**: Collection of three jars
- **v2**: Single ornate jar with swirling dust
- **v3**: Row on shelf, varying fullness
- **v4**: Dust spilling with magical sparkles

## Generation

Generated using `scripts/generate_prop_concepts.py` with Gemini image generation.

```bash
# List all props and variations
python scripts/generate_prop_concepts.py --list

# Generate specific prop
python scripts/generate_prop_concepts.py --prop magic_minivan

# Generate specific variation
python scripts/generate_prop_concepts.py --prop mop_wand --variation 2
```
