February 1, 2026

I need Blender to do what I say. That's the problem I'm solving.

Normally, making a 3D scene means clicking around for hours. Set up objects. Position the camera. Tweak the lights. Render. Hate it. Do it again.

I want to describe what I want and have it appear. So I'm connecting Claude to Blender.

I describe a scene. Claude writes Python code. Blender runs it headless (no GUI). I get a render. If it's wrong, Claude looks at the image and tries again.

That's the theory anyway.

What I've Built So Far

Two scripts to test if this actually works:

poc_create_scene.py â€” Takes a description, generates Blender Python code, creates the scene, spits out a render.

poc_validation_loop.py â€” Renders, checks if it looks right, fixes what's wrong, repeats until it's decent.

What's Hard

Vague words don't translate to code. "Make it more dramatic" means nothing to Blender. I'm trying to build a vocabulary that bridges fuzzy creative direction and specific operations. It's slow going.

Blender has tons of state. The AI needs to know what's already in the scene before it can change anything. Getting that context right is tricky.

Rendering takes forever. Even a simple scene is several seconds. The feedback loop only works if you can iterate fast. I'm using preview renders and simpler geometry to speed things up.

How do you know when it's good enough? I don't have a great answer yet. Right now it's mostly me looking at it and deciding.

What's Actually Working

Some stuff does work:

Basic scene setup from descriptions â€” works pretty well. Camera positioning from shot descriptions â€” surprisingly good. Simple lighting from mood descriptions â€” decent. The feedback loop improving results over passes â€” yes, it helps.

What's Next

Hook up my actual character and environment assets. Figure out animation (right now it's just static scenes). Better quality checking. Running multiple shots at once.

Stuff I Don't Know Yet

How much cinematography knowledge should I bake in? How specific do scene descriptions need to be? What do I do when the creative direction is intentionally vague? When do I just give up and do it manually?

Still figuring it out. I'll post more when I learn more.
