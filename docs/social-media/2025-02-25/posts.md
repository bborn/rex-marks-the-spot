Here's something wild about how we're making this movie.

Today's production update ‚Äî the Scene 1 animatic, the AI tool comparisons, the character tests ‚Äî most of that work was done by AI agents running autonomously.

We use Claude Code (by Anthropic) coordinated through a task system called TaskYou. Each agent gets a task, works in its own git branch, and submits a pull request when it's done. Just like a human developer would.

Today the agents:
- Fixed character consistency issues across storyboard panels
- Set up and configured our GPU rendering infrastructure
- Ran animation experiments across multiple AI tools
